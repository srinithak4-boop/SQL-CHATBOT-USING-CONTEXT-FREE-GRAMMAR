# SQL Chatbot Using Context-Free Grammar

## 📌 Project Overview
SQL Chatbot Using Context-Free Grammar is a mini project that converts English language queries into SQL queries using Context-Free Grammar (CFG). The system allows users to interact with a database using simple English instead of writing SQL commands.

## ✨ Features
- Converts English queries into SQL queries
- Uses Context-Free Grammar (CFG) for parsing
- Supports SQL query generation
- User-friendly interface
- MySQL database integration
